Mise en pratique

Objectif

Acquérir les bons réflexes de manipulation et de qualité des données.

  • Questions
  • Conseils
  • Solutions

  1. Télécharger le jeu de données decathlon disponible dans le package FactoMineR.

  2. Afficher un aperçu du jeu de données grâce à la fonction glimpse. Observer les différents types de variables. Renommer les variables dont le nom commence par un chiffre et intégrer le nom des lignes en tant que variable.

  3. Transformer le jeu de données en format long, de sorte qu’il y ait une ligne par résultat pour chaque épreuve (tout en gardant les autres colonnes).

  4. Identifier si le jeu de données contient des valeurs manquantes.

  5. Représenter graphiquement les distributions de performance sur chaque épreuve. Identifier les athlètes ayant obtenu des résultats atypiques.

  6. Calculer la performance moyenne de chaque discipline.

  1. Certains packages R sont accompagnés de jeux de données exemples. Ils peuvent être chargés à l’aide de la fonction data.

  2. Il est fortement déconseillé d’avoir des noms de variables commençant par des chiffres. Pour renommer plusieurs colonnes en suivant le même motif, utiliser la fonction rename_with. Pour ajouter le nom des lignes comme une variable au jeu de données, utiliser la fonction rownames_to_column.

  3. Pour transformer au format long, utiliser la fonction pivot_longer.

  4. Pour sélectionner des lignes, utiliser la fonction filter.

  5. Au sein d’un ggplot, pour représenter des boxplots, utiliser la fonction geom_boxplot. Les outliers (valeurs atypiques) peuvent être identifiés à l’aide de l’argument outlier.color. Pour afficher le nom des athlètes, utiliser geom_text. Les valeurs atypiques se situent à l’extérieur de l’intervalle correspondant à 1.5 fois l’intervalle inter-quartile.

  6. Pour calculer les moyennes des performances de chaque discipline, utiliser les fonctions group_by et summarise.

## 1. Chargement des données
data(decathlon, package = "FactoMineR")

## 2. Aperçu du jeu de données
glimpse(decathlon)

## Renommage de certaines variables
decathlon <- decathlon %>% 
  rownames_to_column(var = "Athlete") %>% 
  mutate(Athlete = toupper(Athlete)) %>% 
  rename_with(.fn = ~paste0("D", .x), 
              .cols = c(`100m`, `400m`, `110m.hurdle`, `1500m`))

## 3. Passer au format long pour identifier les valeurs manquantes et atypiques.
dec_long <- decathlon %>%
  pivot_longer(cols = starts_with(c("D100m", "Long.jump", "Shot.put", 
                                    "High.jump", "D400m", "D110m.hurdle",
                                    "Discus", "Pole.vault", "Javeline", "D1500m")),
               names_to = "Epreuve",
               values_to = "Performance")

## 4. Identifier les de valeurs manquantes
dec_long %>%
  filter(is.na(Performance))

## 5. Identifier les valeurs atypiques
ggplot(dec_long, aes(x = Epreuve, y = Performance)) +
  geom_boxplot(outlier.color = "red", fill = "lightblue", alpha = 0.6) +
  geom_text(
    data = dec_long %>% group_by(Epreuve) %>%
      mutate(Q1 = quantile(Performance, 0.25),
             Q3 = quantile(Performance, 0.75),
             IQR = Q3 - Q1) %>%
      filter(Performance < Q1 - 1.5 * IQR | Performance > Q3 + 1.5 * IQR),
    aes(label = Athlete),
    hjust = -0.1, size = 3, color = "blue"
  ) +
  theme_minimal(base_size = 13) +
  labs(title = "Athlètes aux performances atypiques")

## 6. Calcul de la performance moyenne par discipline
decathlon %>% 
  group_by(Epreuve) %>% 
  summarise(Perf_Moyenne = mean(Performance))

Mise en pratique

Objectif

Représenter graphiquement les déplacements de joueurs au cours d’un match.

  • Conseils
  • Solutions

  1. Importer le fichier data_GPS_rugby.csv et observer son contenu. Rectifier le nom des colonnes pour correspondre au tableau ci-dessus.

  2. Représenter au moyen d’un graphique les trajectoires des 15 joueurs en utilisant ggplot. Ajouter une couleur différente pour chaque joueur. Explorer différents thèmes.

  3. Représenter les coordonnées x et y (Latitude et Longitude) en fonction du temps pour le joueur 8.

  4. Faire de même pour tous les joueurs.

  1. Utiliser read_delim ou read_csvpour charger le fichier, puis rename pour renommer les colonnes.

  2. Les trajectoires peuvent être représentées sous forme de texte (numéro du joueur) avec geom_text, sous forme de points avec geom_point, sous forme de ligne avec geom_path. Pour colorer les joueurs d’une couleur différente, il faut ajouter l’option colour = player dans la fonction aes. Observer la différence si player reste numérique ou s’il est transformé en facteur (ou character). Le thème graphique peut être modifié en ajoutant + theme_classic() ou + theme_minimal() par exemple à la fin du ggplot.

  3. Utiliser la fonction facet_wrap pour représenter la latitude et la longitude sur des panneaux différents.

## 1. Charger fichier et renommer colonne.
GPS_rugby <- read_csv("Data/data_GPS_rugby.csv")
GPS_rugby <- GPS_rugby %>% 
  rename(num =`...1`, 
         Longitude = x, 
         Latitude = y)

## 2. Graphe des trajectoires
ggplot(data = GPS_rugby, 
       mapping = aes(x = Longitude, y = Latitude,
                     label = player, colour = factor(player))) + 
  #geom_text(size = 6, check_overlap = TRUE) + 
  geom_point() +
  theme_classic() 

## 3. 
GPS_rugby %>%
  filter(player == 8) %>%
  pivot_longer(cols = c(Longitude, Latitude)) %>%
  ggplot() + 
    geom_line(aes(x = frame_id, y = value, col = player), col = "#DD3497") +
    facet_wrap(~name, ncol=1, scales="free") + 
    theme_classic() +
    labs(x = 'Temps (frame 10Hz)',
         y = 'Position (mètre)')

## 4.
GPS_rugby %>%
  pivot_longer(cols = c(Longitude, Latitude)) %>%
  ggplot() +
    geom_line(aes(x = frame_id, y = value, col = factor(player))) +
    facet_wrap(~name, ncol=1, scales="free") + 
    theme_classic() +
    labs(x = 'Temps (frame 10Hz)',
         y = 'Position (mètre)',
         col = "Joueur")

Mise en pratique

Objectif

Extraire ces informations à partir d’une page html.

  • Questions
  • Conseils
  • Solutions

  1. Collecter les tableaux présents sur la page Wikipedia du saut à la perche.

  2. Sachant que le tableau des records masculins annuels correspond au 16ème tableau, l’exporter.

  3. Sachant que le tableau des records féminins annuels correspond au 17ème tableau, l’exporter.

  4. Combiner les deux tableaux et ajouter une variable codant la catégorie de l’athlète (M/F).

  5. Mettre en forme la variable “Marque” qui correspond à la hauteur du saut.

  6. Représenter au moyen d’un graphique l’évolution des marques en fonction des années et de la catégorie ainsi que son lissage.

  1. Utiliser les fonctions read_html et html_table.

  2. Utiliser la fonction bind_rows.

  3. Utiliser les fonctions str_remove, str_replace, str_trim, …

  4. Utiliser geom_smooth pour afficher le lissage.

## 1. Collecter les tableaux
### Lecture du HTML
page <- read_html("https://fr.wikipedia.org/wiki/Saut_%C3%A0_la_perche")
### Extraction des tables HTML ----
tables <- html_table(page, fill = TRUE)

## 2. Export du tableau d'intérêt
records_annuels_M <- tables[[16]] %>% 
  as_tibble() 

## 3. Export du tableau d'intérêt
records_annuels_F <- tables[[17]] %>% 
  as_tibble() 

## 4. Concaténer les lignes des deux tableaux.
records_annuels <- bind_rows(records_annuels_M %>% 
                               mutate(Categorie = "M"),
                             records_annuels_F %>% 
                               mutate(Categorie = "F") %>% 
                               rename(Marque = Hauteur))

## 5. Mise en forme de la hauteur du saut
db_records <- records_annuels %>%
  mutate(
    Hauteur = Marque %>%
      str_remove_all("\\[[^]]*\\]") %>%   # supprime les notes [1]
      str_remove_all("m") %>%             # enlève le "m"
      str_replace(",", ".") %>%           # remplace la virgule
      str_trim() %>%                      # supprime les espaces
      na_if("") %>%                       # remplace chaînes vides par NA
      as.numeric(),
    .keep = "unused"
  )

## 4. Représentation graphique
ggplot(data = db_records, 
       mapping = aes(x = Année, y = Hauteur, 
                     label = Athlète, col = Categorie)) + 
  #geom_text(size=3, colour='blue', check_overlap = TRUE) + 
  geom_point() +
  geom_smooth() +
  ggtitle(label = "Meilleures performances mondiales annuelles", 
          subtitle = "1912 à 2024") + 
  theme_minimal()
